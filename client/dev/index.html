Grunt Build Scaffold

This build scaffold provides some generic static site files, and provides grunt tasks for deploying the site to a testing server, and also packaging for production.

Grunt tasks (commands):

  • grunt run: lints js, processes sass, runs any qunit tests, and copies everything to a 'temp' directory. Runs a server at localhost:8000, whose root is the temp directory.
  • grunt build: lints js, processes sass, runs any qunit tests, combines and minifies js files, copies all files to a 'dist' folder, updates script link blocks to point to concat/minified file.
